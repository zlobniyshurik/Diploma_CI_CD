- **Устанавливает следующие пакеты**:
  - `net-tools`
  - `wget`
  - `nano`
  - `unzip`
  - `python3-firewall`
  - `policycoreutils-python-utils`
- **Ограничивает зону `home` в фаерволе адресами из локальных "серых" IP**
